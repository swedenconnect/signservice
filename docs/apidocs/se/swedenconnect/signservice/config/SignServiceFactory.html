      @Nonnull SignServiceEngineManager createSignServiceEngineManager(@Nonnull SignServiceConfigurationProperties configuration, @Nullable BeanLoader beanLoader, @Nullable BeanRegistrator beanRegistrator) throws Exception
      Creates a SignServiceEngineManager based on the supplied configuration.
      Parameters:
      configuration - the SignService configuration
      beanLoader - the bean loader. If not supplied and the configuration contains bean references the method will fail
      beanRegistrator - a bean registration callback (optional)
      Returns:
      a SignServiceEngineManager instance

      Exception - if the configuration is incorrect and a SignServiceEngineManager can not be created
